# 🛒 Superstore Sales Dashboard & Forecasting (Power BI)

## 📌 Project Overview

This project presents an **interactive Superstore Sales Dashboard** built using **Microsoft Power BI** to analyze business performance and predict future sales trends.

The dashboard provides deep insights into:

* Sales performance 📈
* Profit analysis 💰
* Customer segments 👥
* Shipping trends 🚚
* Sales forecasting 🔮

Additionally, a **15-day Sales Forecasting model** is implemented to help in business decision-making.
